Table

That the table tag opens a table. Each row is defined by tr tag. Each column within a row is defined by td tag. The th tag is used for data headers

There are lots of attributes that tables can use. Some examples are below

  • border - this sets a border for the table tag.
  • width - sets the width of a table or column
  • height - sets the height of a table or column
